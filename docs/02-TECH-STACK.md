# Technology Stack

## Frontend Framework
- **React** 18.3 - UI library
- **TypeScript** 5.8 - Type-safe JavaScript
- **Vite** 5.4 - Lightning-fast build tool

## UI & Styling
- **shadcn/ui** - Accessible component library
- **Tailwind CSS** 3.4 - Utility-first CSS framework
- **Radix UI** - Headless component primitives

## State Management & Data
- **React Router** 6.30 - Client-side routing
- **React Query** 5.83 - Server state management
- **React Hook Form** 7.61 - Efficient form management
- **Zod** 3.25 - TypeScript-first schema validation

## Visualization
- **Recharts** 2.15 - React charting library
- **Lucide React** - Icon library

## Animations & Interactions
- **Framer Motion** 12.23 - Animation library
- **Embla Carousel** 8.6 - Carousel component

## Utilities
- **date-fns** 3.6 - Date manipulation
- **clsx** 2.1 - Conditional className utility
- **tailwind-merge** 2.6 - Merge Tailwind classes
- **Sonner** - Toast notifications
- **next-themes** - Dark mode support

## Development Tools
- **ESLint** 9.32 - Code linting
- **Autoprefixer** - CSS vendor prefixes
- **PostCSS** - CSS transformations

## Why This Stack?

| Tool | Benefit |
|------|---------|
| **Vite** | Instant hot module replacement, fast builds |
| **TypeScript** | Type safety, better IDE support |
| **React + React Router** | Component-based UI with client-side routing |
| **shadcn/ui** | Pre-built accessible components |
| **Tailwind CSS** | Rapid UI development with utility classes |
| **React Query** | Powerful server state management |
| **Framer Motion** | Smooth animations and transitions |
